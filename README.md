
# 📸 MTPC — Photography Portfolio Website  
### A PHP-Based Photography Web Application

MTPC is a simple and responsive photography portfolio web application developed using PHP, HTML, CSS, JavaScript, and MySQL. The project is designed to showcase photography collections, portfolios, and creative works through a clean and visually appealing interface.

This project was developed as a Major Project and can be easily hosted on any standard PHP hosting server or run locally using XAMPP/WAMP/LAMP.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Hazy-Arrow/MTPC-Major-Project.git
```

---

## 2️⃣ Move Project Folder

Copy the project folder into:

```bash
htdocs/
```

(for XAMPP)

---

## 3️⃣ Start Local Server

Start:

* Apache
* MySQL

from XAMPP/WAMP control panel.

---

## 4️⃣ Create Database

* Open phpMyAdmin
* Create a database
* Import:

```bash
database.sql
```

---

## 5️⃣ Configure Database Connection

Update database credentials inside:

```bash
includes/config.php
```

Example:

```php
<?php
$host = "localhost";
$user = "root";
$password = "";
$database = "mtpc";
?>
```

---

# ▶️ Run Project

Open browser:

```bash
http://localhost/MTPC-Major-Project
```
